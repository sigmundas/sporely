# Calibration dialog

Figure 1 placeholder: overview of the Calibration dialog.

Use this dialog to create, review, and activate scale calibrations for microscope objectives and macro setups.

## Main areas

- Top row: choose an objective/profile, create or edit profiles, load calibration images, and export the current view.
- Calibrate from Image: measure a known distance manually or run automatic calibration on a slide image.
- Manual Entry: enter a known scale directly if you already know the value.
- Calibration History: review saved calibrations, mark one as active, or delete outdated entries.

## Typical workflow

1. Select or create the objective/profile.
2. Load one or more calibration images.
3. Run automatic calibration or add manual measurements.
4. Review the result and save the calibration.
5. Set the preferred calibration as active.

## Notes

- The active calibration is the one used when measuring images with that objective.
- If a calibration is already used by images or measurements, the dialog warns before deletion.
