# Prepare Images dialog

Figure 1 placeholder: overview of the Prepare Images dialog.

Use this dialog to review and tag images before saving them into an observation.

## Main areas

- Left panel: add/remove images, reorder them, and switch between field and microscope image types.
- Center preview: inspect the selected image, crop it, rotate it, and check scale-related overlays.
- Right panel: edit image metadata such as objective, contrast, stain, mount, sample type, and notes.
- Bottom gallery: reorder and review all images in the current import batch.

## Typical workflow

1. Add or review the image set.
2. Set image type for each image.
3. For microscope images, assign objective and microscopy metadata.
4. Crop or rotate images where needed.
5. Continue to the observation dialog.

## Notes

- Field images and microscope images share the same dialog but use different metadata fields.
- The bottom gallery can be resized; thumbnail size follows the gallery height.
- Scale-bar calibration here is meant for imported images that already contain a visible scale bar.
