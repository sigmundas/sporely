# New/Edit observation dialog

Figure 1 placeholder: full observation dialog overview.

Use this dialog to create a new observation or edit an existing one.

## Main areas

- Observation Details: date, coordinates, map link, location name, and comments.
- Taxonomy: species/common-name entry, genus/species fields, habitat/substrate/host tabs, and AI suggestions.
- Images: review the images currently attached to the observation.

## Typical workflow

1. Check date, place, and coordinates.
2. Enter the taxon, or pick from the mixed lookup / AI suggestions.
3. Add notes, habitat, and confidence details.
4. Review the attached images.
5. Save the observation.

## Notes

- The `Genus` field can be left empty if the lookup can resolve it from the species/common name.
- Location name lookup will not overwrite a manually edited location name unless you use `Get name`.
- `Edit images` opens the Prepare Images dialog for the same observation.
